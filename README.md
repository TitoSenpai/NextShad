## How to install

[](https://github.com/nobruf/shadcn-landing-page#how-to-install)

1.  Clone this repositoy:

git clone https://github.com/TitoSenpai/NextShad.git

2.  Go into project

 `cd nextshad`

3.  Install dependencies

`pnpm install`

4.  Run project

`pnpm run dev`
